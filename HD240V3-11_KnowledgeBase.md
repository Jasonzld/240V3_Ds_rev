## 1. 产品概述

**产品**：微酸性电解水生成设备控制板（消毒液制备）

**工作原理**：电解稀盐酸溶液生成次氯酸消毒液。MCU 监测电解电流，调节蠕动泵定量补酸以维持目标电流恒定。

**核心控制回路**：检测电流 → 蠕动泵定量补盐酸 → 维持恒流电解 → 生成消毒液

**启停模式**：
- 屏端按键触发
- 红外感应开关触发（类似洗手间感应龙头）
- 外接开关触发
- 支持点动模式和锁存模式

### 工作流程

1. 开进水阀（220V AC，U27 SSR 控制）→ 自来水进入电解槽
2. 霍尔流量计检测进水流量 (EXT_IN_COUNTER, U8.36)
3. 电解槽通电（Q27 开关，24V 软启动：1s 1kHz PWM 0→100% 占空比）
4. MCU 监测电解电流（U9 CC6903 霍尔电流传感器 → U8.14 ADC）
5. 蠕动泵按需定量注入盐酸，调节电流至目标值
6. 达到启停条件后关闭电解槽和水阀

## 2. 系统架构

```
24V DC 主供电 (CN3)
  ├── F1(贴片保险) → VIN
  │     └── U12(EG1192 DCDC) + L2(47uH) → **12V** (系统主功率轨)
  │           ├── U5(BL8032 降压) + L3(10uH) → **+5V** (USB/通信/AMS1117输入)
  │           │     └── U10(AMS1117-3.3 LDO) → **+3.3V** (MCU/Flash/逻辑)
  │           └── U6(BL8032 降压) + L4(10uH) → EXT+5V/S+5V
  │                 └── U17(MT9700 限流) 串联在 S+5V 与 EXT+5V 之间
  │
  └── F2(限流保险) + D36(TVS) → VBUS+
        └── U9(CC6903 电流检测) → V_POWER
              └── Q27(NCEP40T20ALL) → VQ → DJC1 → **电解槽** (24V)
  │
  └── F2(限流保险) + D36(TVS) → VBUS+
        └── U9(CC6903 电流检测) → V_POWER
              └── Q27(NCEP40T20ALL) → VQ → DJC1 → **电解槽** (24V)

220V AC (CN3 独立 Pin)
  └── F3(保险座) → U27(AQH3223A SSR) → 220V-L_OUT → 水阀

电源链: 24V → EG1192 → 12V → BL8032 → 5V → AMS1117-3.3 → 3.3V

电解控制:
  MCU CORE_ONOFF(U8.41) → R27(10Ω) → Q1(AO3400A) → Q25(HSS1N20) → Q27.1(NCEP40T20ALL栅极)
     启动: 1kHz PWM 0→100% 占空比, 持续 1s (软启动)
     运行: 高电平 (Q27 导通, 电解槽工作)
     关闭: 低电平 (Q27 截止)
     注: EXT_PWM 连接器及相关电路(Q18/Q28/Q29)未正式启用

蠕动泵:
  MCU HCL-PUMP(U8.35) → U1(DRV8870 H桥) → 蠕动泵 (定量注盐酸)

传感器:
  U9(CC6903) → U8.14 ADC (电解电流闭环)
  U31(LM358) → U8.15 ADC (电压测量)
  霍尔流量计 (进水流量检测)
  红外感应开关 → CN2.10 → W_SENSOR → U8.53 (高电平触发)

通信:
  U4(SP3232 RS-232) ↔ 大彩7寸串口屏 (TFT接口)
  U16(CH340N USB-UART) ↔ USB1 (编程/调试)
  GH2 ↔ 合宙Air780 4G模块 (选配, UART)
  SWD1 ↔ SWD调试接口
```

## 4. 电源架构

**电源链**：24V → EG1192 → 12V → BL8032 → 5V → AMS1117-3.3 → 3.3V

### 4.1 供电入口

| 输入 | 连接器 | 用途 |
|------|--------|------|
| **24V DC** | CN3 (两个 Pin) | 主板全部低压供电 |
| **220V AC** | CN3 (独立 Pin) | 仅供水阀 (经 U27 SSR) |

### 4.2 电源树

```
24V DC (主供电 CN3)
  ├── F1(JFC2410-1300TS) → VIN
  │     └── U12(EG1192 DCDC) + L2(47uH) → 12V (系统主供电轨)
  │           ├── U5(BL8032) + L3(10uH) → +5V
  │           │     ├── U10(AMS1117-3.3) → +3.3V (MCU/Flash/逻辑)
  │           │     └── S+5V (传感器/上拉)
  │           ├── U6(BL8032) + L4(10uH) → +5V / EXT+5V
  │           │     └── U17(MT9700 限流开关) → EXT+5V (外部)
  │           └── 风扇(Q24) / 继电器等直接 12V 负载
  │
  └── F2(A72-300 限流保险) + D36(SMCJ70A TVS) → VBUS+
        └── U9(CC6903 霍尔电流检测) → V_POWER
              └── Q27(NCEP40T20ALL 功率开关) → VQ → DJC1 → **电解槽 24V**

220V AC (CN3 独立 Pin)
  └── F3(5x20 保险座) → U27(AQH3223A SSR) → 220V-L_OUT → 水阀
```

### 4.3 电源轨

| 网络 | 电压 | 来源 | 主要负载 |
|------|------|------|---------|
| VIN | 24V DC | F1 后 | EG1192 (U12) 输入 |

| +5V | 5V DC | **BL8032 (U5)** 降压 | USB / 通信 / AMS1117 输入 |

## 11. 电解槽驱动子系统

### 11.1 Q27 栅极驱动链

```
MCU CORE_ONOFF (U8.41) 输出 PWM/GPIO
  ├── R27 (10Ω) ──► Q1 (AO3400A) 栅极
  │     └── Q1.3 (漏极) ──► Q25 (HSS1N20) 栅极
  │           └── Q25.3 (漏极) ──► Q27.1 (NCEP40T20ALL 栅极)
  │                                 └── R29 (10K, 上拉到 12V)
  │
  └── R74 (1K, 下拉到 GND) — 关机时确保 Q18/Q1 关断
      └── Q18 (AO3400A) → EXT_PWM.3 (EXT_PWM 连接器未启用)

软启动时序:
  启动: MCU 输出 1kHz PWM, 占空比 0%→100%, 持续 1 秒 (防冲击电流)
  正常运行: 高电平持续 (Q27 导通)
  关闭: 低电平 (Q27 截止)

注: Q18/Q28/Q29 通往 EXT_PWM 连接器的信号均未正式启用
```

### 11.2 功率回路

```
V_POWER (24V, U9 电流检测后)
  └── Q27 (NCEP40T20ALL) ──► VQ (Q27.9 漏极) ──► DJC1.2 ──► 电解槽 (+)
                                                      │
DJC1.1 (VBUS-, 电流返回) ◄────────────────────────── 电解槽 (-)
  └── U9.3/U9.4 (CC6903 IP-)

注: R45 (0.05Ω) 是 EG1192 DCDC 的电流采样反馈, 非电解回路
注: L2 (47uH) 是 EG1192 DCDC 的输出电感 (12V), 非电解回路
```

### 11.3 DJC1 接口 (VH3.96-3P2, 实际 2 Pin)

| Pin | 网络 | 连接 | 功能 |
|-----|------|------|------|
| 1 | VBUS- | U9.3/U9.4 (CC6903 IP-) | 电解电流返回 (经 Hall 传感器) |
| 2 | VQ | Q27.9, R22, R52 | 电解槽 24V 供电 (开关后) |

### 11.4 电流闭环与安全保护

```
电流闭环:
  U9(CC6903 霍尔传感器, 串联在 VBUS+ → V_POWER 路径) 
    → VOUT (经 R23/C53 滤波) → CURRENTSENSOR (U8.14 ADC) 
    → MCU 计算实际电流
  
  实际电流 < 目标-回差 (如 1.75A) → 启动蠕动泵定量补盐酸
  实际电流 > 目标+回差 (如 1.85A) → 停止蠕动泵
  
  目标电流: 1.80A (可设范围 0.5A~5A)
  回差值: 可设 (如 50mA)

安全保护:
  一般上限电流 → 告警 (可设)
  危险上限电流 → 紧急停机 (可设，如 >5A)
  电流错误 → ERR_CUR (Q4 NUD3105) → ERR_CUR1 输出
  流量错误 → ERR_FLOW (Q3 NUD3105) → ERR-FLOW1 输出
```

## 13. 设计注意事项

1. **供电区分**：24V DC 是主板全部供电来源（CN3 两 Pin），220V AC 仅供水阀（CN3 独立 Pin），两者在 PCB 上需保持爬电距离
2. **MCU 型号**：以 BOM 为准 → **SMT32F410RBT6** (BOM MPN 字段 STM32F103RDT6 为采购参考，非实际焊接型号)
3. **电流采样电阻**：R45 (0.05Ω/1210) 和 R57 (0.2Ω/2512) 为大功率电流采样，注意温漂和散热
4. **电解槽软启动**：CORE_ONOFF 信号启动时需输出 1kHz PWM 0→100% 占空比 1 秒，避免冲击电流
5. **红外感应输入**：W_SENSOR (CN2.10, U8.53)，高电平有效，经 U32 施密特触发整形
6. **ESD 全面防护**：所有外部通信口均有 TPD4E05U06DAQR 或 TVS 管保护
7. **未焊接项**：部分 22uF/25V 电容 (C29-C31, C57, C75-C78, C80-C81, C83, C135-C136) 无厂家型号，需确认是否已焊接
8. **4G 模块**：合宙 Air780 为选配，通过 GH1/GH2/GH3 三组 4P Header 连接，UART 通信
9. **SPI Flash**：U13 (W25Q128) 通过 SPI 连接，用于固件/数据存储
10. **BOOT 配置**：SMT32F410 的 BOOT0/BOOT1 引脚需检查配置以确定启动模式
